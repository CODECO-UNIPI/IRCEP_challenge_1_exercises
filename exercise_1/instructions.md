# Άσκηση 1 : Δημιουργία και εκτέλεση Dockerized Python εφαρμογής
</br>

Απλό παράδειγμα Python εφαρμογής μέσα σε Docker.


</br>
</br>

## Περιγραφή

Το script `app.py` υπολογίζει το άθροισμα τετραγώνων των αριθμών **1..N**.  
Ο κώδικας βρίσκεται στον φάκελο `src/`.



</br>
</br>

### Οδηγίες

</br>

#### Δημιουργία φακέλου

</br>

1. Δημιουργήστε έναν φάκελο έργου, π.χ. **docker-python-task/**
2. Γράψτε ένα Python script που να υπολογίζει μια μαθηματική πράξη  
   (π.χ. άθροισμα τετραγώνων, μέσο όρο, τυπική απόκλιση για αριθμούς 1…N).  
   Ο κώδικας βρίσκεται στο αρχείο **src/app.py**.
3. Δημιουργήστε ένα **Dockerfile** που:
   - Χρησιμοποιεί ως βάση μια Python εικόνα (π.χ. `python:3.11-slim`)
   - Αντιγράφει το `app.py` και τα απαραίτητα αρχεία στο container
   - Ορίζει την εντολή εκκίνησης (π.χ. `CMD ["python", "app.py"]`)

> Δείτε το περιεχόμενο του αρχείου **Dockerfile** στο repository.

</br>


#### Δημιουργία Docker Image
</br>

```bash
docker build -t my-python-app .
```
</br>


#### Εκτέλεση του Container 
</br>

Εκτέλεση με προεπιλεγμένη τιμή (π.χ. N=10)

</br>

```bash
docker run my-python-app
```
</br>

#### Αλλαγή της τιμής N

</br>

Εκτέλεση με διαφορετική τιμή N (override)

</br>

```bash
docker run my-python-app python app.py 50
```
